# Lively Gradient Ribbons

Animated gradient ribbons wallpaper for **Lively Wallpaper**, featuring real-time music reactivity and optional clock, weather, and media widgets. Browser canvas keep performance smooth while delivering dynamic visuals that react to audio and user settings.

## Summary

- **Dynamic Visuals**  
  Ribbon reacts to music beats in real time using Lively’s audio API.

- **Lightweight**  
  Built for smooth performance with minimal GPU/CPU usage on modern systems.


## Features

- Live gradient ribbons with smooth browser-driven animation  
- Audio-reactive pulsing, distortion.
- Optional widgets:  
  - Live clock  
  - Weather display  
  - Media info (song title/artist)  
- Full support for Lively Wallpaper’s customization panel  
- Resolution-independent; scales cleanly on multi-resolution settings  


## Installation

### 1. Download the Latest Release
Grab the newest `.zip` from the **[Releases](https://github.com/brkeejp/lively-gradient-ribbons/releases)** section of the repository.

### 2. Unzip
Extract the entire archive to any folder of your choice.

### 3. Install to Lively
Open **Lively Wallpaper → Add Wallpaper → Browse**,  
select the extracted project folder, and confirm.

Lively will automatically detect the configuration and load the wallpaper.
